# Project Euler Solutions 🧮💻

## Overview:
This repository contains my personal solutions to Project Euler problems. They can be found @ https://projecteuler.net/

I use these challenges to work on my problem-solving skills in the world of mathematics and coding.

## Note 
I'm fully aware and respect the fact that the Project Euler community would like solutions to remain private.
I'm making these solutions as a portfolio piece to demonstrate my problem-solving skills.

If you are working through Project Euler -- Do the problems yourself first.

## Repo Structure 
The solutions are solved in Python.

Each script is named to the corresponding problem index value. eg: Problem_001.py 

At the top of each script is the original question followed by my solution code.
